Source: https://www.kaggle.com/datasets/sarveshchhetri/play-tennis-practice-dataset-for-classification/data

The **Play Tennis Dataset** is a synthetic, expanded classification dataset inspired by the classic “Play Tennis” example widely used in machine learning and data mining education.

This dataset models the relationship between weather conditions and the decision to play tennis, making it ideal for practicing supervised learning algorithms. It includes multiple categorical features such as weather outlook, temperature, humidity, and wind strength, with a binary target variable indicating whether tennis is played.

The dataset has been expanded to include a larger number of records while preserving realistic feature combinations and class balance. It is well-suited for beginners and intermediate learners to explore data preprocessing, categorical encoding, probabilistic models, and decision-based algorithms.
